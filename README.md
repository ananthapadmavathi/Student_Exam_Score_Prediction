# Student Exam Score Prediction

## Project Description
This project predicts student exam scores based on the number of hours studied using Linear Regression.

## Algorithm Used
Linear Regression

## Technologies Used
- Python
- Google Colab
- Scikit-learn
- Pandas
- Matplotlib

## Working Process
1. Load the dataset
2. Train the Linear Regression model
3. Predict the exam score based on study hours
4. Display prediction results

## Example
If a student studies 5 hours, the model predicts the expected exam score.
